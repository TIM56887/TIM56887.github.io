# i18n ally 套件 無法編輯 出現 Writing in readonly mode is disabled. 錯誤
## 造成沒辦法編輯的原因：
使用`js、ts`儲存文字檔案

![使用js、ts儲存文字檔案範例](../asset/i18n_ally.png)
## 解決方法：
> 目前唯一的解決方法只有把檔案換成json或yaml，由於js,ts是動態的，要更改裡面的內容非常困內，目前沒有其他方法
